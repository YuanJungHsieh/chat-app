# chat-app
### 使用 Node.js, Express, Socket.io 實作即時聊天網站

#### 功能：
- 建立使用者名稱
- 加入/ 創建聊天室
- 傳遞即時文字訊息
- 分享所在位置

#### 網站連結：https://leohsieh-chat-app.herokuapp.com
